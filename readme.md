# Pomodoro Timer

A feature-rich, beautifully designed Pomodoro timer application with multiple themes, customizable timers, and focus assistance tools. Built with vanilla HTML, CSS, and JavaScript.

![Pomodoro Timer](screenshot-placeholder.png)

## ✨ Features

### 🎯 Core Timer Functionality
- **Multiple Timer Presets**: Classic Pomodoro (25/5), Quick Focus (15/5), Deep Work (45/15), Micro Sessions (10/3), 52/17 Method
- **Custom Durations**: Set your own study and break times
- **Visual Progress Bar**: See your progress at a glance
- **Pause/Resume/Stop**: Full control over your sessions
- **Session Persistence**: Timer state saves automatically when you close the browser

### 🎨 12 Beautiful Themes
Choose from a variety of carefully crafted color schemes:
- Dark Ocean (default)
- Warm Cozy
- Light Minimal
- Sunset
- Forest
- Midnight
- Mystic Berry
- Neon Dream
- Coffee
- Ocean Breeze
- Mystic Purple
- Sunflower

### 🆘 Focus Assistance
- **"I'm Stuck" Panel**: Get help when focus feels impossible
  - "Just do 2 more minutes" - Small commitment to build momentum
  - "Take a movement break" - 1-minute reset break
  - "Write down what's hard" - Brain dump feature
- **Brain Dump**: Capture distracting thoughts during work sessions

### 🔊 Sound Notifications
- Custom sound uploads for study and break transitions
- Toggle sounds on/off
- Visual and haptic (vibration) feedback where supported

### 📱 Display Options
- **Minimal Mode**: Distraction-free timer display
- **Break Screen**: Full-screen break notifications
- **Visual Timer**: Color-coded progress tracking

### 💾 Data Persistence
- All settings saved to browser's localStorage
- Timer state preserved across sessions
- Custom sounds remembered

## 🚀 Getting Started

### Installation
1. Clone this repository or download the HTML file
2. Open `index.html` in any modern web browser
3. Start focusing!

No build process, no dependencies—just open and use.

## 📖 How to Use

### Basic Usage
1. **Select a preset** from the options (Classic, Quick Focus, etc.)
2. Click **START** to begin your focus session
3. Work until the timer ends
4. Enjoy your break when the break screen appears
5. Repeat!

### Customizing Your Experience
- **Adjust durations** in Settings panel
- **Change themes** from the 12 available options
- **Upload custom sounds** for study/break transitions
- **Toggle minimal mode** for distraction-free work

### When You're Stuck
1. Click **"I'M STUCK"** when you need help focusing
2. Choose from three supportive options
3. Use the brain dump feature to clear your mind
